Decisions captured

  • ✓ Shape: T3 navy rectangle, top-left
  • ✓ Sold out: S3 navy strip across middle of image
  • ✓ Stacking: I7 single tag with caps + italic inside (typographic contrast)
  • ✓ Scarcity badges (Last one / Few remaining) — dropped. Catalogue is naturally scarce; badging it would be wallpaper
  • ✓ Stock count: PDP only, never on cards
  • ✓ Handmade alone: italic Cormorant inside a navy box, no caps
