Stripe receipts are NOT in this preview. Stripe sends its own receipt + invoice PDFs directly (the dark-navy "Receipt from ORLO Store" email). Customisable only via Stripe Dashboard (logo + brand color). Separate from these templates.

One template, every email

All customer emails use the same customerEmail() shell — the difference between them is just the copy passed in (titleEn, bodyEn, bodyAr, infoBoxEn, ctaUrl/ctaText). Same for the 2 admin emails using adminEmail(). When you approve the v2 design, swapping the template file in one commit upgrades all 11 simultaneously.

Function signatures stay identical, so every caller (webhook.js, ship-order.js, return flows, auth flows, etc.) works without changes.
